What is Okyaku?

Okyaku is an AI-Native CRM Platform designed from the ground up with artificial intelligence at its core.

Key Differentiators

100% Self-Hosted

Complete data sovereignty. Your data never leaves your infrastructure.

Zero API Costs

Local AI inference with Ollama. No recurring AI service fees.

Privacy-First

No data sent to external services. Full compliance ready.

AI-Native Design

Built with AI from day one, not bolted on as an afterthought.

Getting Started

Prerequisites

  • Python 3.11+
  • Node.js 18+
  • PostgreSQL 14+
  • Ollama

Quick Start

# Clone repository
git clone https://github.com/gnoobs75/Okyaku.git

# Pull AI models
ollama pull llama3.1:8b
ollama pull nomic-embed-text

# Run setup
setup.bat

# Start development
start-dev.bat
