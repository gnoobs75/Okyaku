## POST-PRODUCTION NOTES

### Recommended Editing

1. **Transitions:** Use simple cross-dissolves between scenes (0.3-0.5 seconds)
2. **Zooms:** Add subtle zoom-ins when highlighting specific UI elements
3. **Callouts:** Add arrow or circle callouts to draw attention to key features
4. **Lower thirds:** Add scene titles as lower-third graphics during transitions
5. **Cursor highlight:** Consider adding a yellow/red circle around cursor for visibility

### Audio

- **Background music:** Soft, upbeat corporate music at 10-15% volume under narration
- **Sound effects:** Subtle click sounds when clicking buttons (optional)
- **Narration:** Clear, professional voice at consistent volume

### Thumbnails

Create a thumbnail showing:
- Okyaku logo
- Dashboard screenshot (partial)
- Text: "Quick Start Guide"
- Play button overlay

## RECORDING TIPS

1. **Browser:** Use Chrome or Edge for best rendering
2. **Extensions:** Disable all browser extensions that might show notifications
3. **Screen recorder:** OBS Studio (free) or Camtasia (paid) recommended
4. **Microphone:** Use a decent USB microphone for clear narration
5. **Multiple takes:** Record each scene separately for easier editing
6. **Cursor visibility:** Enable cursor highlighting in your screen recorder
7. **Pause between scenes:** Leave 2-3 seconds of silence between scenes for editing
